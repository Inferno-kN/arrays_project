начну с того, какие команды я использовал
1) git add - инициализация системы гит
2) Писал имя и почту
git config user.name "Ваше_Имя"
git config user.email "Ваш_Email"
3) написал одну функцию для одномерного массива
и сделал коммит
git commit 
4) каждый раз добавлял по функции и также их все коммитил
git commit
5) после каждого добавления одной функции и после коммита, я просматривал историю добавлений
git log
6) ещё просматривал изменения что добавлял 
git diff
7) когда дело дошло до отката, выполнил все команды такие как
git restore --staged
8) и делал сам откат
git checkout
но откат почему то не откатывался до последнего изменения
